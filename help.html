LZ-log help

Starting the application

  1. Make sure you have installed Java Runtime Environment (JRE) version 1.7 or later.

  2. Important! Extract the program into a directory which path does not contain Unicode characters or empty spaces (only ASCII). Make sure also that the directory name is not super long.

    • Example of a good dir: C:/log/Lz-log

    • Example of bad dirs: C:/my logs/lz-log; C:/Щраус логове/lz-log; C:/Program Files/lz-log

  1. Start the program by using the startup.bat (for Windows) or the starup.sh (for Linux)



Application settings

Radio CAT settings

Select the comport and baud to which you have connected your radio.

Keyer

Currently supported keyers are Winkeyer and DTR/RTS pin keying.

My Callsign

The callsign that you will be using during the contest.

Function keys

Each F1 to F9 key sends a Morse code message. If you want to include your call inside the message use the format: {mycall}

Contest Rules

Specify how often you can work a station. The default value is 30mins (1800secs).

Time to next QSO

You can specify a timeout value after which the callsign will not be visible in the Window “Time to Next QSO”. This should be a negative value.

Misc

“Enable quick callsign entry” - if this is enabled you need to specify the default prefix that will be added to all callsign that you enter inside the entry window. Default value is LZ.

“Send leading zeros as “T”” - if enabled 005 will be sent as tt5. However, 105 will be not sent as 1t5.

“Enter sends message” - if enabled pressing enter in the Entry window will also send certain Morse code messages.



Entry window

The entry window is where the callsign and the serial number are written. If the “Enable quick callsign entry” setting is activated there is no need to enter the prefix of the station. For example instead of LZ1ABC you can enter 1ABC.

  • CQ – when this radio button is selected you are in run mode (i.e. calling CQ). When you work a station in this mode it will not be added to the Bandmap window.

  • S&P – Search and Pounce mode (e.g. you are searching for stations). When you work a station in this mode it will be automatically added to the Bandmap window. If you want to add a station manually to your Bandmap you need to enter the stations callsign and press F11 while being on the frequency of the station.

Function keys

  • F1-F9 – transmits a predefined Morse code message. You can change the predefined messages from the Tools->Settings.

  • F11 – manually adds a callsign to your Bandmap.

  • F12 – wipe all the fields clean. Also you can use the key combination ALT+W or CTR+W

Shortcut keys

  • ; - Sends TU message and enters the contact in log

  • ' - Sends hos call and exchange

  • ALT+W or CTR+W – same as F12



Radio/Keyer connection

Radio

Set the desired com port and baud rate in the Settings and then press the Connect button. Then navigate to the file describing the communication protocol of your radio.

  • For Kenwood rig: kenwood/kenwood.py

  • For K3: elecraft/elecraft.py

  • For Yaesu: yaesu/yaesu.py

  • For Icom rig select the file: icom/icom.py

For Icom rigs only: Each Icom model has its own unique address and when communicating with the rig over the serial connection the program needs to use this unique address. For this purpose you need to edit the file pyrig/icom/icom.py. Open the file using the a text editor and edit the line CIV_ADDRESS = 0x5C The value 0x5C is good for Icom 756PRO. If for example you have IC-7000 you need to edit it so that CIV_ADDRESS = 0x70 After this is done open the program and connect using the Connect button.

For Yaesu rigs only: In ft-450d (and maybe other Yaesu rigs) there is a bug in the CAT protocol so that after switching between VFO A and B the rig first send the mode for the new VFO and then sends the info that the VFO has changed. This leads to inability of this software to display the mode for the current VFO reliably. If you see that the current Mode in the program does not represent correctly the Mode of the radio you have to options: A) Toggle between Modes on your radio. B) Turn polling on.

Polling: If you transceiver does not send data automatically when changing frequency or mode you need to set the “Enable polling” check box. However, for many transceiver you can enable the rig to send this status data automatically so that polling is not needed. Enabling is done through the internal menu of the rig. For example for Icom transceivers the menu is called CI-V Transceive and should be set to ON.

Keyer

Set the desired keyer type and com port in the Settings and then press the Connect button.

Note: If you want to use the same com port for the keyer and the radio - make sure that you first connect to the radio and then to the keyer.

Note: If you radio supports sending Morse code over the CAT you don't have to connect to the keyer (connecting to the radio is enough). Radio that I know to support sending Morse code over the CAT are Elecraft K3 and Kenwood TS-590.



Bandmap window

Shows which station is located on which frequency. There are two ways to add a station to the bandmap

  1. Work the station while in S&P mode.

  2. Add the station pressing the Spot button – currently F11.

    • Stations added by spotting them (F11) will have a star in front of them. This will help you discern between manually added and added while working the station.

In the Bandmap there can be only one unique station spotted and one unique station while working this station.

If you can't see the complete callsign inside the band map you can:

  • Use the settings “Enable quick callsign entry” from the settings window. This will remove the LZ from the callsign.

  • Use a display with greater resolution.

  • Hide the frequency columns by setting “Show freq” to false. I am always hiding this column – it is kinda useless.

  • Reduce the number of columns: this will make your columns wider.



Misc window

  • Continuous CQ – if selected once you press the F1 key the program will transmit periodically a CQ message. The period is specified in milliseconds.

  • F1 jumps to CQ frequency – if selected when F1 is pressed the program will change the frequency to the last stored CQ frequency. CQ frequency is stored by pressing the button “Set CQ freq”

  • Keyer settings – with the buttons UP(page up) and Down(Page down) you can set the speed of the keyer.
